TinyMce_ImageUpload
===================

ImageUpload plugin for TinyMce

<br/>
It works at version 4.0.28


init:
-----
```javascript

<script>
tinymce.init({
	    ...,
	    imageupload_url: '/tinymceDemo/upload/cms/', // thie directory to save upload file 
		plugins: [
			"... imageTony ..." 
		],
		toolbar: "... | imageTony", 
		...
 });
 
</script> 
 
```
